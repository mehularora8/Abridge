Abridge is a one-of-its kind service that can be used to summarise large texts and annotate lectures. Textbooks and other learning sources tend to be extremely long and wordy. Abridge eliminates the need for such texts and transcripts and provides short bullet points for them. It has been developed with the purpose to be primarily applied in the education sector. However, it can also be used in other professions like journalism, law, bureaucracy etc. It has been implemented using Natural Language Processing libraries. With Speech to Text support, Abridge can also be used by deaf and limb amputated students to annotate lectures. Try it out now!
